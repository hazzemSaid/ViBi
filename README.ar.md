## 🚀 عن ViBi

ViBi هو تطبيق اجتماعي قوي مفتوح المصدر مبني باستخدام Flutter و Supabase. يجمع التطبيق بين تفاعل وسائل التواصل الاجتماعي الحديثة ونظام مخصص للأسئلة والأجوبة، مما يتيح للمستخدمين التواصل ومتابعة صناع المحتوى المفضلين لديهم وطرح الأسئلة (سواء بهوية مجهولة أو معلنة).

## ✨ الميزات الأساسية

- **🔐 مصادقة قوية**: تسجيل دخول آمن عبر البريد الإلكتروني و Google OAuth باستخدام Supabase.
- **👤 ملفات شخصية غنية**: ملفات قابلة للتخصيص مع صور رمزية، نبذة تعريفية، وروابط اجتماعية متكاملة.
- **📥 صندوق وارد ذكي**: مساحة مخصصة لاستقبال وإدارة الأسئلة.
- **💬 أسئلة وأجوبة مجهولة**: إمكانية طرح الأسئلة والإجابة عليها مع خيار إخفاء الهوية.
- **📱 خلاصة ديناميكية**: خلاصة اجتماعية لحظية تعرض المحتوى من الأشخاص الذين تتابعهم.
- **📸 القصص (Stories)**: مشاركة محتوى اجتماعي مؤقت يستمر لمدة 24 ساعة.
- **🔍 بحث متقدم**: العثور على المستخدمين ومحتوى محدد بسرعة وكفاءة.
- **🔔 إشعارات**: ابقَ على اطلاع دائم مع الإشعارات الفورية في الوقت الفعلي.

## 🏗 المعمارية

تم بناء المشروع باستخدام مبادئ **Clean Architecture** لضمان القابلية للتوسع والاختبار والصيانة:

- **طبقة العرض (Presentation Layer)**: منطق واجهة المستخدم وإدارة الحالة باستخدام `Bloc/Cubit`.
- **طبقة المجال (Domain Layer)**: منطق العمل الخالص، الكيانات (Entities)، وتعريفات حالات الاستخدام (Use Cases).
- **طبقة البيانات (Data Layer)**: تنفيذ المستودعات (Repositories) ومصادر البيانات (GraphQL, Supabase, Rest).

*تمت إعادة هيكلة المشروع مؤخراً لفرض فصل صارم بين المسؤوليات، وتوحيد معالجة الأخطاء باستخدام `Either` وتجميع المكونات البرمجية المشتركة.*

## 🛠 التقنيات المستخدمة

- **الواجهة الأمامية**: [Flutter](https://flutter.dev/) & [Dart](https://dart.dev/)
- **الخلفية**: [Supabase](https://supabase.com/) (PostgreSQL, Storage, Real-time)
- **إدارة الحالة**: [Bloc/Cubit](https://pub.dev/packages/flutter_bloc)
- **طبقة API**: [GraphQL](https://graphql.org/) (عبر [Ferry](https://pub.dev/packages/ferry))
- **التوجيه (Routing)**: [GoRouter](https://pub.dev/packages/go_router)
- **معالجة الأخطاء**: [Dartz](https://pub.dev/packages/dartz) (أنماط البرمجة الوظيفية)

## 🛠 البدء بالعمل

### المتطلبات الأساسية

- Flutter SDK (أحدث إصدار مستقر)
- Dart SDK
- مشروع Supabase (للتطوير المحلي، ستحتاج لمفاتيح API الخاصة بك)

### التثبيت

1. **استنساخ المستودع**:
   ```bash
   git clone https://github.com/hazzemSaid/ViBi.git
   cd ViBi
   ```

2. **تثبيت التبعيات**:
   ```bash
   flutter pub get
   ```

3. **إعداد البيئة**:
   قم بإنشاء ملف `.env` في المجلد الرئيسي مع بيانات اعتماد Supabase و Google OAuth الخاصة بك.

4. **تشغيل التطبيق**:
   ```bash
   flutter run
   ```
